## Local Setup & Run Guide

To get this project running locally on your machine, follow these steps:

### 1. Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- npm (comes with Node.js)

### 2. Installation
Clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/Jetshree/demo-website-jetshree.git
cd demo-website-jetshree/nsb-website

# Install dependencies
npm install
```

### 3. Running the Development Server
Once all packages are installed, start the local server:

```bash
npm run dev
```

The terminal will provide a local URL (usually `http://localhost:3000`). Open this link in your browser to view and interact with the website!
