NSB is built with **CMake** and depends on a few well-known system libraries. Follow the steps below to get it compiled and installed on your machine.

---

## Prerequisites

Install the following packages before building NSB:


| Package | Purpose |
| :--- | :--- |
| **CMake** | Configure and build the project |
| **Pkg-Config** | Package configuration (required on macOS) |
| **Redis** | Message payload store (database) |
| **Abseil** | Protobuf support and logging |
| **Protobuf** | Message serialization |
| **yaml-cpp** | Parse YAML configuration files |
| **hiredis** | C client library for Redis |
### macOS: via Homebrew

```bash
brew install cmake pkg-config abseil protobuf yaml-cpp redis hiredis
```

### Linux: via apt (Ubuntu/Debian)

```bash
sudo apt-get install -y cmake pkg-config libprotobuf-dev protobuf-compiler \
  libyaml-cpp-dev libhiredis-dev redis-server
```

> **Note:** Abseil may need to be installed from source on older Linux distros. See the [Abseil GitHub](https://github.com/abseil/abseil-cpp) for instructions.

---

## Build

Create a `build` directory at the root of the repository and run CMake:

```bash
mkdir build
cd build
cmake ..
cmake --build . --clean-first
cmake --install .
```

After installation, the library files, headers, and daemon binary are placed under:

```
[your/install/path]/nsb/
  ├── bin/        # nsb_daemon executable
  ├── include/    # C++ headers
  └── lib/        # Shared libraries
```

---

## Uninstallation

To remove NSB from your system, choose one of the following:

**Option A**: Delete the install directory:

```bash
rm -rf [your/install/path]/nsb
```

**Option B**: Use the install manifest generated by CMake:

```bash
xargs rm < build/install_manifest.txt
```
